## 📌 1️⃣ Key Performance Indicators (KPI) 📊

The project used structured KPIs to monitor schedule and cost performance. The primary KPI was the **Baseline Execution Index (BEI)**, which measured planned vs completed tasks to track adherence to the baseline schedule. Performance thresholds (Green ≥99%, Yellow 95–98%, Red ≤94%) were used to identify delays and trigger corrective actions.

Financial performance was tracked through comparing actual spend with the planned budget (±including tolerance). Supporting KPIs included **ROI, Budget Utilization, and Time-Phased Budget**

## ⚙️ 2️⃣ Process 🔄

The project followed a structured lifecycle beginning with defining scope, objectives, and budget. A **Work Breakdown Structure (WBS)** and baseline schedule were created to guide execution.

The team conducted **financial analysis, risk assessment, and mitigation planning**, while continuously monitoring progress using BEI and financial tracking. The project concluded with formal closure and documentation.

## 📊 3️⃣ Data Insights 📈

The project generated key insights through structured analysis. The total budget of **$143,400** was allocated across labor, non-labor, and risk buffer, demonstrating strong financial planning.

Risk analysis identified **financial mismanagement** as the highest-impact risk, highlighting the importance of cost control. Resource allocation totaled **857 labor hours**, showing efficient team utilization.

Site selection insights were derived from evaluating supply chain efficiency, logistics, workforce availability, energy access, and regulatory approvals.
